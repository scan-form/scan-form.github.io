ScanForm

Memeriksa lembar jawaban pilihan ganda secara online.

Mulai

Fitur

  • Gratis.
  • Langsung pakai. Tidak perlu daftar. Tidak perlu login.
  • Lembar jawaban tidak memerlukan kertas khusus. Cukup dengan kertas fotokopi biasa.
  • Hasil pemeriksaan lembar jawaban dilengkapi dengan analisis butir soal yang komprehensif.
  • Hasil pemeriksaan dan analisis butir soal disimpan ke file dalam format MS Excel.
  • Privasi. Proses pembacaan lembar jawaban dilakukan secara lokal. Data hasil pemeriksaan tidak dikirim ke server.

Cara Penggunaan

  • Cetak lembar jawaban ini di kertas A4.
  • Perbanyak lembar jawaban menggunakan mesin fotokopi.
  • Isi lembar jawaban dengan menggunakan pinsil 2B.
  • Pindai (scan) lembar jawaban yang sudah diisi.
  • Periksa file hasil pindai di halaman web ini.
