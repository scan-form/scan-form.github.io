ScanForm - Keterangan

ScanForm Pilihan Ganda adalah aplikasi web gratis untuk memeriksa jawaban pilihan ganda. Form lembar jawaban bisa anda download di link ini. Cetak ke kertas ukuran A4, dan perbanyak form lembar jawaban tersebut dengan mesin fotokopi.

Untuk bisa menggunakan aplikasi ini, lembar jawaban yang sudah diisi harus dipindai (scan) pada resolusi 300 dpi dengan format file PNG black and white (bukan grayscale atau berwarna). Anda juga perlu menyiapkan kunci jawaban pada kertas form tersebut untuk dipindai.

Cara Menggunakan

Keterangan untuk setiap isian pada aplikasi ini adalah sebagai berikut:

  • Jenis pekerjaan:
    • Baca form dan beri nilai: membaca lembar jawaban dan memberi nilai berdasarkan kunci jawaban yang anda berikan.
    • Hanya baca form: hanya membaca lembar jawaban, tanpa memberi nilai.
  • Jumlah pertanyaan: cukup jelas.
  • Kunci jawaban: pilih file hasil scan lembar jawaban yang telah anda isi dengan kunci jawaban.
  • Form jawaban: pilih 1 sampai 50 file scan lembar jawaban yang akan diperiksa.
  • Daftar nama dan ID (opsional): Jika anda menginginkan hasil pembacaan form memiliki kolom nama, maka anda perlu menyediakan file daftar nama dan nomor ID dalam format file CSV. Untuk membuat file tersebut, anda bisa menggunakan program MS Excel, atau Google Spreadsheet dan menyimpannya dalam format CSV. File daftar ini harus terdiri dari dua kolom, di mana kolom pertama harus berisi nomor ID, dan kolom ke dua berisi nama. Baris pertama berisi judul kolom. Anda bisa melihat contohnya di link ini.

Klik tombol Proses untuk memulai proses pembacaan form. Tunggu beberapa saat sampai aplikasi ScanForm menampilkan hasilnya. Anda bisa menyimpan hasil pembacaan form ini ke dalam file MS Excel dengan mengklik tombol Save Hasil. Jika jenis pekerjaan yang anda tentukan adalah Baca form dan beri nilai, maka akan ada tombol Save Analisis Soal yang bisa anda klik untuk menyimpan data hasil analisis butir soal (dalam format file MS Excel).

Apabila ada file scan yang tidak bisa dibaca, ScanForm akan menampilkan nama filenya untuk anda periksa.

ScanForm bisa membaca sampai 50 lembar jawaban dalam satu kali proses pembacaan. Apabila anda memiliki lebih dari 50 lembar jawaban, maka anda bisa memproses sisanya dengan mengisi kembali isian Form jawaban dengan sisa file scan tersebut, dan selanjutnya tinggal klik tombol Proses.

Tombol Reset bisa anda klik apabila anda ingin memulai lagi proses dari awal.

Untuk mencoba, silakan download contoh file berikut. Ekstrak file tersebut ke satu folder, kemudian gunakan file-file tersebut sebagai input aplikasi web ini.

Web Browser

ScanForm memerlukan salah satu dari web browser berikut:

  • Google Chrome versi 74 atau yang terbaru.
  • Firefox versi 66 atau yang terbaru.
